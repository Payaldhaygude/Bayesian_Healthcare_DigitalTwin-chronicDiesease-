How it works
5-step probabilistic pipeline from raw data to personalised risk
1

Data Preprocessing

Raw Pima Indian Diabetes dataset values (Glucose, BMI, Blood Pressure, Insulin, Age) are converted into meaningful categorical nodes: Diet, Exercise, Sleep, Stress, BMI Level, Insulin Sensitivity.

2

Causal Graph Construction

A Directed Acyclic Graph (DAG) encodes causal relationships. Sleep → Stress, Diet → GlucoseLevel, Exercise → InsulinSensitivity → GlucoseLevel. This is causal structure, not just statistical correlation.

3

Bayesian Parameter Learning

Conditional Probability Tables are learned using the BDeu Bayesian estimator with Laplace smoothing — handling the 3:1 class imbalance that caused the original "always low risk" bug.

4

Variable Elimination Inference

Given your lifestyle evidence, Variable Elimination computes the exact posterior: P(GlucoseLevel = High | Sleep=Poor, Diet=HighCarb, ...). This is exact probabilistic inference.

5

Digital Twin Output

The result is displayed as risk percentages, gauge charts, factor bars, and personalised recommendations — all updating live as you change the patient profile.
